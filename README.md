This is the repository for the official **[Arhi Games](https://arhigames.com)** Homepage. 
The homepage will show all my *commercial* projects and some personal projects the world didn't see yet.

> [!CAUTION]
> This site is still in active development, but already available on arhigames.com. There's still more to come!

## Why
With growing projects & more and more people asking about what I already did and who I am, I decided a Homepage / Portfolio website would be quit useful. Additionally I could use this to apply to various different companies.

## The experience
Firstly, this is first website targeted to be publicly available on a public domain, not just available as Open-Source Software via a docker compose.

Secondly, this is the first time a website of mine just included frontend, not anything with a backend, just a *static* html homepage.

Overall it was pretty fun to develop this website, altough, I usually don't like to develop frontend websites as much, I actually enjoyed developing this one. As I usually develop video games with UE5, I am familiar to a very different style of programming, which I, this time also applied to this project. More in __Techstack__

## Techstack
This app was developed using React and TypeScript, now many people might ask rightfully: "Why would anybode create a static webpage using a heavy weight frontend framework like React?!". And to be honest, I asked that question to myself too. But creating websites in VanillaJS isn't fun for me at all, therefore I decided to use a framework. 

Choosing a framework actually was a good choice, because of the choice I could create a website more easily that builds itself just from a simple .json file, when I want to add a new project, I don't go to the code, but I add the project in the .json file and it's already on the website. Additionally, when I add blogs & post later on, it will be easier to do so in React than in VanillaJS
